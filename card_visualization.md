# Koleksi Kartu Gacha — Legenda Kerajaan Nusantara

Koleksi 100 kartu karakter, makhluk mitologi, dan senjata pusaka bertema legenda
kerajaan dan cerita rakyat Indonesia (Majapahit, Singhasari, Sriwijaya, Pajajaran,
Mataram Kuno, dunia pewayangan, hingga legenda rakyat Nusantara). Setiap kartu
dilengkapi lore singkat, skill gacha, dan prompt image generation yang detail.

## Rules:
1. All Images must have same frame pattern
2. Each Rarity cagtegory must have different color frame
3. All Images must be in PNG format
4. All Images must be in portrait orientation
5. All Images must be in 3:4 aspect ratio

| Rarity | Jumlah Kartu |
